Prerequisites

An undergraduate-level understanding of Database Design concepts.

Objectives

This course is primarily designed for undergraduate students. In this course you will be learned how to design, administrate, and querying relational databases. In this course we focus on Oracle Database and we use Oracle LiveSQL which is a web-based and free platform for using and trying Oracle Database. The topics which will be covered are :

  • Database Design & Implementation
    • Review Relational Concepts
    • Review Semantic Modelling ER/EER
    • Review Normal Forms & Normalization
    • Using Oracle SQL, DDL Commands
    • Database Design Best Practices & Design Patterns
  • Writing Queries & Database Applications Programming
    • Review SQL, DML Commands
    • Review Techniques for Writing Queries
    • Introduction to PL/SQL(Static SQL, Dynamic SQL)
    • Introdcution to Java Database Connectivity (JDBC)
    • Using OJDBC11 & Java15 to Write DB App
